A-Shop 사용자 관리 시스템

아래는 ashop_users 컬렉션의 문서 구조입니다:

{
  uid: "firebase-auth-uid",
  email: "user@ashop.com",
  username: "사용자이름",
  displayName: "사용자이름", 
  phone: "010-1234-5678",
  tier: "bronze",
  points: 1000,
  status: "active",
  signupMethod: "email" | "github",
  emailVerified: true/false,
  marketingAgreed: { email: true, sms: false },
  createdAt: "타임스탬프",
  updatedAt: "타임스탬프", 
  lastLogin: "타임스탬프"
}

기능 요약

  • 사용자 인증: Firebase Authentication 연동
  • 등급 시스템: 브론즈 ~ 다이아몬드 등급
  • 포인트 관리: 적립 및 사용 내역 추적
  • 마케팅 동의: 이메일/SMS 수신 동의 관리
  • 활성 상태: 사용자 계정 활성/비활성 관리

데이터 필드 설명

필드명 설명
tier 사용자 등급 (bronze, silver, gold, platinum, diamond)
signupMethod 가입 방법 (email 또는 github)
marketingAgreed 마케팅 수신 동의 정보
